Easy on the couch -A seven day training course for couchdb

What is couchdb?

couchdb is a document oriented database, written using a highly concurrent and fault tolerant programming language:erlang, that comes with an easy to use web api. Data is stored as JSON. An http api and a data format that is universally understood makes couchdb one of the most easy to use database there is.

couchdb imposes no restrictions on your database schema. You can change your documents to adapt to your changing business requirements. And to top it off couchdb provides a server side javascript environment that you can use to implement unique ways to extract your data, transform your documents, validate their properties, execute unit tests with real data and even do things like rendering a template. Truth is once you really start to get a hang of couchdb you will notice that the boundaries between a database and an application server start to blur. You start doing things that you never thought would even be possible with a database.

What can I do with it?

Since data is stored as documents in couchdb any problem that can be represented in terms of documents would be a good fit for couchdb. Thus real world businesses that involve a lot of documents would find themselves right at home in couchdb. You can develop solutions for:-

  • Legal industry
  • Real estate industry
  • Education industry
  • Medical industry
  • Publishing industry
  • Banking industry
But these are just the obvious use cases. Once you start to understand the powerful incremental map/reduce you will see that it can be used to build anything from recommendation engines to content management systems.
