Usage

Initial tests showed that around 300-500 MB of ram is needed for all the services to run on idle. [Of course we'll need more ram than that because there would be sockets involved that can shoot up the ram usage.]

  1. Build openresty with --with-stream parameter [./configure --with-stream]
  2. Configure maximum incoming connections for mongodb
  3. Install java
  4. We need glibc library for mongodb sudo apt-get install libc6
  5. Install expect (sudo apt install expect) to give automatic answers to command prompts during nodebb installation.. [No need for expect as we can redirect the answers from a file]
